Conference Information

Information Collection/Data Governance

How do we develop policies and practices for better epidemiological data collection and for sharing data with the appropriate authorities?

Date

September 28, 2016

Time

10am - 12pm EST

Participate
Problem Description

Each online session is focused on a specific problem, that has been fleshed out into a briefing to describe the root causes of the problem, current approaches in different countries, and lessons learned from other domains.
